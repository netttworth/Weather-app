A simple desktop weather application built with Python and PyQt5. Search any city and instantly get the current weather conditions, temperature, and a description with an emoji that matches the weather.

Features:

Search weather by city name
Displays current temperature
Weather description with a matching emoji
Powered by the OpenWeatherMap API

Built With

Python
PyQt5
OpenWeatherMap API

Getting Started
Prerequisites
Make sure you have Python installed. Then install the required dependencies:
bashpip install pyqt5 requests
API Key Setup


How to Use:

Launch the application
Type a city name into the search bar
Press Enter or click the search button
View the current weather, temperature, and description
